## Setup & Installation

1. Clone the repository:

```bash
git clone https://github.com/yazan-manaldeen/tic-tac-toe.git
cd tic-tac-toe
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
ng serve
```

Visit `http://localhost:4200` in your browser to see the game in action.

---

## Deployment

Build for production:

```bash
ng build
```

Deploy the `/dist` folder to your preferred hosting platform (GitHub Pages, Netlify, Vercel, etc.).
